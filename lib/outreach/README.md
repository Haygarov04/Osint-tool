# outreach/ — Фаза 5 (автоматичен аутрийч)

Стъб за по-късна фаза. Тук ще влязат модули за:

- Конфигурируеми „offer"-и и шаблони за съобщения.
- AI персонализация с Claude (уникално съобщение спрямо бизнеса).
- Multi-step последователности (follow-up 1/2/3).
- Изпращане по имейл с deliverability (SPF/DKIM/DMARC, warmup, дневни лимити).
- Проследяване (изпратено/доставено/отворено/отговорено/bounce/отписан).
- Compliance: unsubscribe, GDPR/CAN-SPAM, черен списък.

Ще ползва `Lead` от `lib/types.ts`; ще добави нови Redis структури за
Campaign / Message / Event.
