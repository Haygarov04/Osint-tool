# outreach/ — AI Персонализация (xAI / Grok)

Вече имаме работеща AI генерация на съобщения:

- `lib/xai/client.ts` — директен клиент към https://api.x.ai/v1 (grok-4.3)
- `lib/outreach/generateMessage.ts` — силно prompt engineering + контекст от лийда
- `app/api/ai/generate-message` — API endpoint
- UI: глобална оферта + ✨ бутон на всеки ред в таблицата + модал с копиране

## Как се ползва

1. Добави `XAI_API_KEY` в `.env.local`
2. В UI въведи какво продаваш (напр. "Модерен уеб сайт + SEO")
3. Натисни ✨ до лийд → Grok генерира уникално съобщение, базирано на:
   - Липса / стар сайт
   - Индустрия и град
   - Рейтинг, tech stack и т.н.

- Автоматично изпращане + tracking
